# 🚀 AI-Assisted Spec-Driven Development (SDD) Demo

Welcome to the **AI-Assisted SDD Demo** repository! This project was entirely generated by an AI coding assistant based on a strict **Spec-Driven Development** prompt.

The repository serves as a showcase of how modern software engineering can leverage AI to autonomously architect, scaffold, and implement a decoupled full-stack application (Spring Boot + React Vite) starting from a single, well-defined Markdown prompt.

## 📁 Repository Structure

- `prompt.md` - The original set of constraints and instructions provided by the human architect.
- `SDD.md` - The professional Software Design Document generated by the AI before writing any code.
- `demo.backend/` - The generated Java 17 + Spring Boot 3.x RESTful API.
- `demo.frontend/` - The generated React 18 + Vite frontend application.

## 🛠️ Technology Stack

| Module | Technologies |
|---|---|
| **Backend** | Java 17, Spring Boot 3.2.x, Spring Data JPA, H2 In-Memory DB, Maven |
| **Frontend** | React 18, Vite, Axios, React Router |
| **Architecture** | RESTful JSON API, strict decoupled layers, CORS |

## 🚀 Quick Start

Both projects are independent. To run the full stack locally on your machine, follow these steps:

### 1. Run the Backend
The backend runs an embedded H2 database and exposes the API on port `50001`.
```bash
cd demo.backend
mvn spring-boot:run
```

### 2. Run the Frontend
The frontend connects to the backend and runs a Vite dev server on port `8088`.
```bash
cd demo.frontend
npm install
npm run dev
```

Navigate to [http://localhost:8088](http://localhost:8088) to explore the live application.

## 🤖 The AI Workflow

This project proves that with the right orchestration and validation workflow, an AI can handle the lower-level implementation details autonomously. The human acts as the **Director (DT)** and **Architect**, while the AI executes the blueprint. 

Feel free to explore the code, test the validation rules, and see how the UI and API interact perfectly.
